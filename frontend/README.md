# Frontend

This folder contains the frontend application files.

## Structure

- `templates/` - HTML templates (Jinja2)
- `static/` - Static assets
  - `style.css` - Main stylesheet
  - `app.js` - JavaScript functionality
  - `logo/` - Logo images
  - `qr/` - Generated QR codes

## Templates

All HTML templates are in the `templates/` folder:
- `base.html` - Base template
- `index.html` - Home page
- `user_*.html` - User-related pages
- `doctor_*.html` - Doctor-related pages
- `hospital_*.html` - Hospital-related pages
- `emergency*.html` - Emergency pages

## Static Files

Static files (CSS, JS, images) are served from the `static/` folder.
